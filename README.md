# Rotten Tomatoes (rottentomatoes)

Rotten Tomatoes is the leading movie and TV review aggregation platform, providing Tomatometer scores, Audience Scores, critic consensus summaries, and review snippets for movies and television shows. The Rotten Tomatoes API enables approved partners to access Tomatometer and Audience scores, critic and audience reviews, box office lists, movie details, cast information, DVD release data, and search capabilities. API access is managed through the Fandango Developer Network and requires an approved application and a paid license agreement starting at $60,000 annually.

## APIs

### Rotten Tomatoes API

The Rotten Tomatoes API v1.0 provides access to movie and TV review aggregation data including Tomatometer scores, Audience scores, critic consensus, critic reviews, box office lists, in-theater listings, upcoming releases, DVD rental data, cast information, and movie search. The API is available to approved partners via the Fandango Developer Network and requires a paid license.

- **Human URL:** [https://developer.fandango.com/rotten_tomatoes](https://developer.fandango.com/rotten_tomatoes)
- **Base URL:** `http://api.rottentomatoes.com/api/public/v1.0`
